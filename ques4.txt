Medium 2 : Maximum Alternating Sum

Prompt:
You're given an array containing N integers. 
You can rearrange this array however you like.
Find the maximum possible value of the alternating sum
after rearrangement.

Sample Input 1:
3, {1,2,3}

Sample Output 1:
4

Sample Input 2:
4, {5,5,5,5}

Sample Output 2:
0
